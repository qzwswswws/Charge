# P4 通道组合 Pilot 总结

## 1. 文档目的

本文档用于整理 `P4` 通道组合 pilot 的阶段性结果。  
该实验的核心目标不是重新定义主线，而是回答一个对硬件设计和论文论证都很关键的问题：

> 在低通道左右手二分类任务下，`C3/C4` 是否真的是最合适的电极组合？

本次结果属于：

- `2-class`
- `5` 个代表性被试 pilot
- 面向通道选择的横向比较实验

因此，它非常适合写入论文中的“电极选择合理性”或“硬件-算法协同设计”部分。

## 3. 实验设置

本轮 pilot 采用如下设置：

- 数据集：`BCI Competition IV 2a`
- 被试：`S1, S3, S5, S8, S9`
- 任务：左右手 `2-class`
- 窗口大小：`window_size = 8`
- 训练轮数：`250 epochs`
- 随机种子：`42`

比较对象包括：

- `c3c4`
- `c3cz`
- `czc4`
- `c3czc4`
- `c1czc2`

其中：

- `c3c4` 复用了已有双导二分类 baseline 的对应被试结果
- 其余组合为本轮新跑结果

## 5. 关键结论

### 5.1 三导中，`C3/Cz/C4` 明显最强

在本轮比较中，`c3czc4` 获得了最高的平均最佳准确率和平均测试准确率：

- `Avg Best = 0.854167`
- `Avg Aver = 0.783645`

相对 `C3/C4` 双导 baseline，其提升为：

- `Best +10.42` 个百分点
- `Aver +8.61` 个百分点

这说明：

- 如果硬件允许从 `2` 个电极放宽到 `3` 个电极，优先加入 `Cz` 是非常有价值的。
- 从生理上看，这也符合 `Cz` 对运动相关中央区活动具有补充信息价值的预期。

### 5.2 双导中，`Cz/C4` 在本轮 pilot 上略优于 `C3/C4`

在仅保留 `2` 个电极的设置下，本轮 `5` 被试结果显示：

- `czc4` 的 `Avg Best = 0.775000`
- `c3c4` 的 `Avg Best = 0.750000`

对应提升约为：

- `Best +2.50` 个百分点
- `Aver +1.09` 个百分点

这说明：

- `C3/C4` 并不是唯一值得考虑的双导组合
- `Cz/C4` 至少在本轮 pilot 中显示出一定竞争力

但需要保持克制表述：

- 当前只是 `5` 被试 pilot
- 还不足以直接推翻 `C3/C4` 作为双导默认方案的地位

### 5.3 `C3/Cz` 的表现不够稳定

`c3cz` 的 `Avg Best` 略高于 `c3c4`，但 `Avg Aver` 反而略低：

- `Best +1.25` 个百分点
- `Aver -0.94` 个百分点

这说明：

- `C3/Cz` 可能能在部分被试上拿到较高峰值
- 但整体稳定性不如 `C3/C4` 或 `Cz/C4`

### 5.4 `C1/Cz/C2` 虽然较强，但更适合作为辅助对照

`c1czc2` 的表现也很突出：

- `Avg Best = 0.816667`
- `Avg Aver = 0.730583`

但从论文叙事和工程实现上看，它更适合作为：

- “中线邻近三导也有潜力”的辅助证据

而不是首选主线，因为：

- 当前主线一直围绕 `C3/C4/Cz`
- `C1/C2` 在论文整体叙事中的历史连续性稍弱

## 6. 对论文的直接价值

这轮 `P4` 的结果可以支撑三类表述：

### 6.1 电极选择不是拍脑袋确定的

现在可以更有底气地写：

- `C3/C4` 的选择并非任意，而是基于 central motor 区的经典侧化逻辑
- 同时，通道组合 pilot 进一步表明，加入 `Cz` 后性能确实能显著提升

### 6.2 如果硬件预算允许，三导方案明显更优

这轮结果实际上给出了一个很清楚的工程建议：

- 极简双导方案可以维持可用性能
- 但若允许增加到 `3` 导，应优先考虑 `C3/Cz/C4`

### 6.3 双导最终方案仍需谨慎表述

尽管 `Cz/C4` 在本轮 `5` 被试 pilot 中略优于 `C3/C4`，但这里不建议现在就把论文结论改写成：

- “双导最佳方案是 `Cz/C4`”

更稳妥的写法是：

- `C3/C4` 是当前主线双导方案
- `Cz/C4` 在 pilot 中显示出竞争力，值得作为后续复核对象

---

## 7. 推荐下一步

如果继续围绕 `P4` 往下走，最值得做的不是再铺更多组合，而是：

1. 将 `czc4` 扩展到 `9` 被试  
   这样可以验证它是否真的稳定优于 `c3c4`

2. 将 `c3czc4` 扩展到 `9` 被试  
   这样可以把“三导是否明显优于双导”从 pilot 结论升级为更正式的证据

3. 不建议继续大范围新增太多组合  
   因为 `P4` 的核心价值已经足够清楚：`Cz` 很重要，尤其在三导条件下
