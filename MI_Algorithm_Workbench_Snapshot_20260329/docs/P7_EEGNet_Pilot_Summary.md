# P7 EEGNet 最小经典对照实验总结

## 1. 文档目的

本文档用于固定 `P7` 中最小经典对照实验的结果，并回答一个直接影响论文完整性的问题：

> 在与当前主线协议一致的前提下，经典轻量 EEG 深度模型 `EEGNet` 的表现如何？它能否改变当前已经形成的主线结论？

## 2. 实验定位

本轮 `EEGNet` 工作并不是要展开完整的大规模方法学比较，而是做一个“最小但够用”的经典对照：

- 补齐“不是只和自己比”的证据
- 选择论文里最关键的两条任务做横向比较
- 以尽量小的额外实现成本，为主线方法的合理性提供参照

因此，这轮实验应定位为：

- `P7` 的最小经典对照
- 面向论文完整性的补充实验
- 与当前 `Conformer` 主线并列、但不替代主线的比较证据

## 5. 结果解读

### 5.1 可以明确支持的结论

- `EEGNet` 在当前统一协议下能够稳定跑通，并形成可用的经典深度学习对照。
- `EEGNet` 并没有推翻当前主线结论。
- 在 `C3/C4 / 2分类` 这一论文当前最关键的落地任务上，`Conformer` 依然优于 `EEGNet`。

### 5.2 更细的比较判断

- `22导 / 4分类` 上，`EEGNet` 和 `Conformer` 的差距不大，说明主线方法并不是建立在“对手极弱”的前提下。
- `C3/C4 / 2分类` 上，`EEGNet` 的结果是可用的，但没有超过当前主线的 `Conformer` baseline。
- 因而，`EEGNet` 更适合作为“经典轻量深度模型参照”，而不是替代当前主线方案。

### 5.3 应如何在论文中表述

更稳的表述是：

- `EEGNet` 作为经典轻量深度模型，在当前协议下可获得与主线方法同量级的全通道性能。
- 但在低通道双导二分类任务上，当前 `Conformer` 主线仍表现出更好的整体识别能力。
- 因而，`P7` 的价值在于补齐横向参照，而不是改写论文主结论。

## 6. 论文中的推荐表述

可以直接写成：

> 为补充横向对照，本文引入经典轻量脑电深度模型 `EEGNet`，并在与主线方法一致的 `BCI Competition IV 2a` 协议下开展最小经典对照实验。结果表明，在 `22导 / 4分类` 任务上，`EEGNet` 与主线 `Conformer` 模型表现处于同一量级；而在 `C3/C4` 双导左右手二分类任务上，`EEGNet` 的平均最佳准确率为 `72.22%`，低于主线 `Conformer` 的 `75.00%`。这说明当前主线模型的优势并非来自缺少经典参照，而是在低通道关键任务上仍保持了更优的整体性能。
